📦 ESTRUTURA DE PRODUTOS IDENTIFICADA
═══════════════════════════════════════════════════════════════════════════════

Tabela Principal: bling_produtos_detalhes_distribuicao
   • id → ID único do produto
   • nome → Nome do produto
   • codigo → SKU
   • 475 produtos cadastrados

Relacionamento com Vendas:
   bling_pedidos_venda_detalhes_itens_ecommerce.itens_produto_id
      → bling_produtos_detalhes_distribuicao.id

Query Funcional:
   SELECT 
     p.nome,
     SUM(i.itens_quantidade) as qtd_vendida,
     SUM(i.itens_valor * i.itens_quantidade) as receita
   FROM bling_pedidos_venda_detalhes_itens_ecommerce i
   JOIN bling_produtos_detalhes_distribuicao p ON p.id = i.itens_produto_id
   GROUP BY p.id
   ORDER BY receita DESC
