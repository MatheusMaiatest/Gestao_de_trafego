════════════════════════════════════════════════════════════════════════════════
                    COMO RESPONDER SUAS PERGUNTAS
════════════════════════════════════════════════════════════════════════════════

❓ "Qual campanha converteu mais vendas?"
   → Vá em: Tráfego Pago > Campanhas
   → Ordene por "Conversões" ou "Receita"

❓ "Qual região compra mais?"
   → Esta funcionalidade usa a integração com clientes
   → O sistema já cruza campanhas com endereços dos clientes

❓ "Qual tipo de cliente compra mais?"
   → O sistema identifica automaticamente PF vs PJ pelo CPF/CNPJ
   → Análise disponível nos dados de produtos vendidos

❓ "Qual o ROI de cada campanha?"
   → Veja a coluna "ROAS" na aba Campanhas
   → ROAS > 2.0 = Excelente
   → ROAS > 1.0 = Positivo
   → ROAS < 1.0 = Prejuízo

════════════════════════════════════════════════════════════════════════════════
                         CAMPO CHAVE
════════════════════════════════════════════════════════════════════════════════

🔑 id_campaign (tabela: produtos_vendidos_tray_ecommerce)

Este é o campo que conecta TUDO:
   Campanha → Produto Vendido → Pedido → Cliente → Região

SEM este campo, seria impossível fazer atribuição de vendas!
COM este campo, sabemos exatamente qual campanha gerou qual venda!
